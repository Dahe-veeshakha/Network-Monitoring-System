# Project Overview

## Introduction

Network Monitoring System is a Python-based network discovery framework developed for Raspberry Pi and Linux environments.

The objective of the project is to continuously monitor local networks, discover connected devices, fingerprint hosts, and maintain a real-time inventory using passive and active discovery techniques.

## Core Features

- Passive Packet Discovery
- Active ARP Scanning
- Nmap Discovery
- Device Fingerprinting
- SQLite Logging
- Rich Terminal Dashboard
- Multi-threaded Scanning
- Confidence-based Detection

## Target Platforms

- Raspberry Pi
- Ubuntu
- Kali Linux
- Debian
